Projects

Dog Breed Classification (Dec'22 -- Jan'23)
The dog breed classification project involves building a machine learning model that can accurately identify the breed of a dog from a given image. The model will be trained on a dataset of labeled dog images, and will use deep learning techniques to recognize the unique features and characteristics of each breed.
Skin Cancer Classification (Mar'23 -- May'23)
Created a model that can accurately classify a disease of 9 classes having 9012 images in total and used Augmentation using Keras ImageDataGenerator to equalize the dataset. Developed a deep CNN model using DenseNet121 architecture that achieved 99 percent accuracy with 89 percent precision in classifying images.
